# BCD Clock Counter

## Overview

This project was completed as a part of **ECE 35800: Intro to VHDL** at Purdue University Fort Wayne. It describes a seven segment display with a variable number of digits. It takes a DATA vector and uses a multiplexer to display the data as a number. A directed test bench is included for simulation purposes.

## Tools

**ModelSim** was used to compile and simulate the project.
